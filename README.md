# Mango Leaf Disease Detection

A deep learning-based mobile application to detect mango leaf diseases using image recognition and provide early treatment suggestions for farmers.

---

## Features
- Image-based disease detection  
- Deep Learning model (CNN)  
- Identifies diseases like Anthracnose, Powdery Mildew, etc.  
- Suggests basic treatment methods  
- Cross-platform mobile app (Flutter)  

---

## Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV  
- Flask (API)  
- Flutter (Frontend)
